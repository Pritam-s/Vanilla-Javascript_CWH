Arrays and Objects :-

  • The object class represents one of JavaScript’s data types.
  • It is used to store various keyed collections and more complex entities.
  • Objects can be created using the Object() constructor.
  • we have two types of values used in JavaScript- primitive and reference.
  • To be more simple, either we create objects or primitive data types.

Examples of primitive data types :-

  1. let myVar = 34;
  2. let myVar2 = "string";
  3. let myVar3 = true;
  4. let myVar4 = null;
  5. let myVar5 = undefined;

Note :- Apart from primitive data types, all the other are objects.

Objects :-

  • The object class represents one of JavaScript’s data types.
  • Objects can be created using object() constructor

Arrays :-

  • There is a special kind of object known as 'arrays'
  • The JavaScript Array class is a global object that is used in the construction of arrays; which are high level, list-like objects
  • An array is a special variable, which can hold more than one value at a time.
  • We can also create arrays with the help of a 'new' keyword.
  • arrays created using 'new' keyword should have '( )' as the constructor instead of '[ ]', otherwise it throws an error.

If else (else-if ladder) & switch case statements :-

Open up the browser console.

  1. if (condition) { } else-if (condition) { } else { }
  2. switch (key) { case value: break; default: break; }
    • switch case statement - Switch case statement should have a break after every case else the program gets executed till the last case.
    • if the value of any of the cases doesn't matches, the default values is chosen.
    • in emmet if we type "switch", we get two switch. Select the second one since it have a default syntax.
